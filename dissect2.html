Dissection II: Public and Private Methods

"Private" Methods

With the proper use of function scope, we can create "inner" or "private" methods.

function bar () { function foo () { alert("foo"); } } alert(typeof foo); // undefined alert(typeof bar.foo); // undefined

"Public" Methods

We can also create public methods (in quite a few ways)

function bar () { this.foo = function () { alert("foo"); }; // instance member method, as an assignment, it must end with semicolon } bar.bad = function() { alert("bad!"); }; // "function" member method (available with or without instantiation, similar to static) bar["baz"] = function () { alert("baz!"); }; // "associative array" style assignment of a member alert("bar.foo is: " + typeof(bar.foo)); alert("bar.bad is: " + typeof(bar.bad)); alert("bar.baz is: " + typeof(bar.baz)); var b = new bar(); // create an instance to access instance members alert("b.foo is: " + typeof(b.foo));
