Skeleton: Prototype and Constructor, the Bones of Javascript

Every object (and thus every function) has one corresponding prototype object, even if it has not been explicitly set.

By explicitly creating a prototype object, or modifying the implicitly created prototype, we can add members (properties and methods).

window.bar = function bar () { }; alert("implicit prototype: " + bar.prototype.toSource()); bar.prototype = { foo: function () { alert("FOO!!"); } }; // an anonymous object, which is equivalent to using Object constructor alert("explicit prototype: " + bar.prototype.toSource());

We can also iterate over properties in a prototype. for (var i in bar.prototype) { alert(i + ": " + bar.prototype[i].toSource()); }

Iterating over an object's members (including properties and methods like bar.foo) is the heart of the jQuery.extend method.

Simplified:

window.extend = function extend(source,target) { for (var name in source) { target[name] = source[name]; } }; var source = { foo: "foo!" }; var target = { bar: function () { alert("bar!"); } }; extend(source,target); for (var name in target) { alert(name + ": " + target[name].toSource()); }
